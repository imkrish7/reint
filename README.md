

## Getting Started

## First, clone the repo
```cmd 
git clone  https://github.com/imkrish7/reint.git
```

## After cloning, get into the directory
```cmd
cd reint or "what you have named"
```
## Install the dependencies
```cmd
npm install
# or
yarn
```

## Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
